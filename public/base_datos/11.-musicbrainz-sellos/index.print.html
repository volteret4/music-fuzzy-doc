11. Musicbrainz sellos

Este script obtiene y almacena información detallada sobre sellos discográficos desde MusicBrainz y fuentes adicionales como Wikipedia y Discogs.

Subsections of 11. Musicbrainz sellos

Argumentos

Clave Descripción Opcional
db_path Ruta a la base de datos SQLite No
cache_path Ruta al directorio de caché Sí
log_level Nivel de registro (DEBUG, INFO, WARNING, ERROR) Sí
mb_sellos Configuración específica del módulo No

Credenciales

Variable Descripción Opcional
MUSICBRAINZ_APP Nombre de la aplicación para usar la API de MusicBrainz No
MUSICBRAINZ_VERSION Versión de la aplicación No
MUSICBRAINZ_EMAIL Correo electrónico para identificación en MusicBrainz No
DISCOGS_TOKEN Token de autenticación para la API de Discogs Sí

Archivos

Tipo Ubicación Típica Descripción
Base de Datos Ruta especificada por db_path Archivo SQLite donde se almacena la información
Caché cache/mb_sellos/ Directorio para almacenar datos descargados desde MusicBrainz
Logs logs/mb_sellos.log Archivo de registro con nivel configurable (por defecto: INFO)
